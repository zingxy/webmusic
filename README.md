# WebMusic
从0开始的项目
## 技术栈
`vite` `Vue3` `vue-router` `pinia`
## 主要功能
- 基本的音乐器

## 解决的问题
- 移动端布局
- Audio 音乐快速切换导致的`DOMException`
- 歌词解析

## 优化
###  交互优化 
- 使用Better-scroll达到更好的滚动效果
- 自定义Vue指令实现loading效果
- 组件切换过度效果

### 性能优化
- 图片懒加载
- 路由组件懒加载
- 歌词缓存
